Name Forge

A simple yet effective Markov Chain Name Generator

Features

This project is centered around an algorithm working in three distinct steps that I will describe here.

Trigram Counter

The names are parsed one after the other, each one being read as a succession of blocks of three characters called trigrams. Each trigram is then counted and stored inside a 3D matrix.

Matrix Probailizer

Now that the count matrix is finished, the algorithm converts it into a probability-based matrix called a transition matrix, where each trigram encountered during the parsing will be assigned a probability to transition towards any other compatible trigram.

Name Generator

Where the magic happens : generate new names based on the probabilities stored inside the transition matrix.

About This Project

The origin of this project comes from the first time I played Dark Souls 3 several months ago. When I created a new character, I stared blankly at the screen, realizing I didn't have any idea how to name my character, and when I searched on the web for new ideas I was frustrated to not be able to generated names following the same rules as Dark Souls names.

Then I remembered the concept of markov chain I encountered several years ago, and it clicked in my head : I want to create my own Name Generator using this algorithm

Therefore, Name Forge is aimed to any other gamers, writers, table top RPG player that want to create new names based on preexisting ones.

Created as a Portfolio Project for Holberton School.
